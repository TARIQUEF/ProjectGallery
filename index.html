Farsiana's Project Gallery

To Do App

Cute, pocket-sized to-do list application developed using JavaFX for a user-friendly GUI and SQLite with JDBC for persistent data storage and database connectivity. Users can add, delete, and complete tasks of their choosing through an intuitive interface, with task data stored and retrieved from an SQL database.

Stuck in CandyLand

3D first-person shooter game developed using C# and Unity. The game involves 3D projectile shooting using raycasting, enemy AI, and a low-poly-Esque 3D environment. Players navigate through the level using WASD movement and the enemy AI adapts to the player's actions- providing a challenging experience. Utilizes C# scripting to implement gameplay mechanics and polish the overall user experience.

Minesweeper

Graphical implementation of the classic Minesweeper game created with JavaFX. The app's grid-based interface allows players to reveal cells while avoiding hidden mines. Players can selectively reveal cells as the game delivers feedback.

Slime Game

3D Pokemon inspired multiplayer game developed using C# and Unity. Players can catch, train, and battle with their own virtual monsters in a 3D world. The game features real-time multiplayer capabilities, allowing players to interact with each other or against computer AI. Implements gameplay mechanics using C# scripting and utilizes Photon servers to facilitate online multiplayer functionality.
