Stuck in CandyLand

This is a 3D first-person shooter game developed using C# and Unity. The game involves 3D projectile shooting using raycasting, enemy AI, and a low-poly-Esque 3D environment. Players navigate through the level using WASD movement and the enemy AI adapts to the player's actions- providing a challenging experience. Utilizes C# scripting to implement gameplay mechanics and polish the overall user experience.
